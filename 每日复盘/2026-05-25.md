# Git 与 Skills 入门

今天算是正式上手 Claude Code 了。先是搞明白了 skills 这东西怎么用，搜了一圈 Excel 相关的技能，发现分两种——一种是直接读写 xlsx 文件的，另一种是通过 MCP 实时操控打开着的 Excel 窗口，区别还挺大的。最后把官方的 xlsx 和 excel-mcp 都装上了，全局的那种。

接着折腾 GitHub，以前没搞过。生成了 SSH 密钥，踩了好几个坑：host key 验证失败、公钥粘贴格式不对 GitHub 不认，来回弄了好几遍才搞定。然后 git init、写 gitignore、commit、push 一条龙，把自己项目 311 个文件推上去了。中间还搞明白了 npm 能通但 git 不通是因为 npm 走了国内镜像，git 直连被墙，最后用 SSH 绕过去的。

总的来说今天从零到有，把工具链搭起来了，还顺手写了份 README 当记录。
